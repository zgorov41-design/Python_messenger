Данный мессенджер работает только одной сети Wi-Fi либо проводного подключения.
Если человек подключен к другой сети вы не сможете с ним связаться.
Сначала распакуйте .zip архив в любое место
Также для работы месенджера необходим пиотн версии не ниже 3.12.3
 Для запуска на Windows:Откройте Cmd(Win+R,введите cmd) и Выполните
 следуйщие команды (установка пакетов):
 установите venv(виртуальную среду):
 Перейдите в директорию
    cd (путь до папки messager)
 Устаноите сам venv:
    python3 -m venv venv
 Запустите venv (также в cmd)
   venv\Scripts\activate
 После активации в начале строки терминала появится префикс (venv):
 (venv) "Имя пользователя":~$
 И устаноите пакеты:
        pip install -r (path to requirements.txt)
 например:(pip install -r /home/"username"/Desktop/messenger/requirements.txt)
 После просмотрите список утсановленых пакетов: pip list
 Там должны присутвовать следуйщие пакеты:Flask==2.3.3
                                          Werkzeug==2.3.7
 Если у вас присутсвуют другие пакеты     Flask-SocketIO==5.3.6
 это нормально                            gevent==24.2.1
                                          gevent-websocket==0.10.1
Далее выйдите из дирекотрии:
   cd
 После успешной утсановки всех пакетов (до сих пор в Cmd) введите:
   python3 "путь до файла" app.py 
 Далее просто перейдите по адресу http://192.168.1.76:5000/
 (закрывать терминал и app.py нельзя)
 Для проверки зайдите с двух разных устройств или просто откройте две вкладки
 в браузере.
 К сожалению пока что мессенджер работает только при обновлении страницы
 и повторного входа
