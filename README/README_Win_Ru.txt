Данный мессенджер работает только одной сети Wi-Fi либо проводного подключения.
Если человек подключен к другой сети вы не сможете с ним связаться.
Сначала распакуйте .zip архив в любое место
Также для работы месенджера необходим пиотн версии не ниже 3.12.3
 Для запуска на Windows:
 Дважды нажмите на install requirements.bat
 После успешной утсановки всех пакетов 
 Дважды нажмите на run.bat
 Далее просто перейдите по адресу указаному в Cmd
 (закрывать Cmd нельзя)
 Для проверки зайдите с двух разных устройств или просто откройте две вкладки
 в браузере.
 К сожалению пока что мессенджер работает только при обновлении страницы
 и повторного входа
